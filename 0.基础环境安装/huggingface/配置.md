# 一在线在服务器上使用Client下载
## 1. conda中安装,yml可能已经安装好了
  pip install huggingface_hub
## 2. 登录 
  huggingface-cli login
## 3. 去设置里面创建一个Read的token
  xxxx
## 4.下载
  huggingface-cli download openai/whisper-medium
